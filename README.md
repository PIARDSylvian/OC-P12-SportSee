# SportSee Front

## Set up server

create .env.local file
add in file :
BACKEND to set backend ULR & port

## How to start dev server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## How to start prod server
```bash
npm run build
npm run start
# or
yarn build
yarn start
# or
pnpm build
yarn start
```
